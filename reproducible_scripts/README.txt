These are standalone reproducibility scripts for Spatchat SDM.

Export behavior:
- The app copies scripts relevant to SDM preprocessing/modeling/postprocessing.
- It also bundles:
  - repro_config.json (session settings and active predictors)
  - copied input tables used by the run
  - outputs/ tables and rasters generated by the run
  - outputs/predictor_effect_plots/*.png when predictor-response plots were generated

Usage:
- Scripts are distributed as .txt so reviewers can inspect them quickly.
- Rename a script to .py to execute directly.
- Scripts are plain Python and do not import Spatchat app modules or LLM components.
- Run scripts from this folder.

Scripts by stage:
- preprocessing_reproduce.txt
- logistic_regression_reproduce.txt
- issa_reproduce.txt
- postprocessing_reproduce.txt
- postprocessing_effect_plots.py

Recommended execution order:
1. preprocessing_reproduce.txt
2. logistic_regression_reproduce.txt or issa_reproduce.txt
3. postprocessing_reproduce.txt

Common dependencies:
- pandas, numpy, scikit-learn, rasterio, scipy, statsmodels (optional for some uncertainty outputs)
